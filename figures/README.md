# Figure File Organization

This folder stores all plots and charts generated in the project, organized by functionality.

### `algorithm_comparison/` - Algorithm Comparison

- `line_charts_comparison.png` - Line chart comparison

### `environment_plots/` - Environment Simulation Plots
Stores various state plots of the home energy management environment:
- `ev_soc_and_price.png` - EV SOC and price plot
- `ess_power_and_pv.png` - ESS power and PV generation plot
- `ac_power_and_temp.png` - AC power and temperature plot

- `reward_components.png` - Reward components plot
- `plot_summary.png` - Plot summary
- `p2.png` ~ `p10.png` - Various detailed environment state plots

### `experiment_results/` - Experiment Results
Stores result charts from various comparison experiments:
- `carbon_emission_analysis.png` - Carbon emission analysis plot
- `strategy_carbon_comparison.png` - Strategy carbon emission comparison plot
- `seasonal_comparison.png` - Seasonal comparison plot

### `model_evaluation/` - Model Evaluation
Stores charts related to model performance evaluation:
- `arbitrage_analysis_*.png` - Arbitrage analysis plots
- `arbitrage_breakdown_*.png` - Arbitrage breakdown plots
- `arbitrage_statistics_*.png` - Arbitrage statistics plots
- `evaluation_plots_*.png` - Evaluation result plots
- `violation_rate_*.png` - Violation rate plots

### `training_progress/` - Training Progress
Stores charts from model training process:
- `td3_returns_*.png` - TD3 algorithm return curves
- `td3_losses_*.png` - TD3 algorithm loss curves

## üîß Usage Instructions

1. **Auto-generation**: All plots are automatically generated by corresponding Python scripts and saved to the appropriate folders
2. **Path Configuration**: Save paths in each script have been unified to this structure
3. **File Naming**: Timestamp format is `YYYYMMDD_HHMMSS`, ensuring file uniqueness
4. **File Management**: Old scattered folders (`save/`, `save_pictures/`, `evaluation_results/` plots) have been integrated into this structure
